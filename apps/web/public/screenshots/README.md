# PWA Screenshots

This directory should contain screenshots for the PWA installation prompt.

## Required Screenshots

Take screenshots of key pages (540x720px for mobile):
1. screenshot1.png - Homepage or product listing
2. screenshot2.png - Course page or dashboard

## Guidelines

- Use 9:16 aspect ratio (portrait)
- Recommended size: 540x720px or 1080x1440px
- Show the most appealing features
- Include actual content, not placeholder text
- Maximum 8 screenshots
