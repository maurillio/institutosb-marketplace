# PWA Icons

This directory should contain the app icons for PWA installation.

## Required Icons

Generate the following icon sizes from your logo:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png (Apple Touch Icon)
- icon-192x192.png (Android)
- icon-384x384.png
- icon-512x512.png (Android splash)

## How to Generate

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/favicon-converter/

## Design Guidelines

- Use a simple, recognizable logo
- Ensure good contrast
- Make it work on both light and dark backgrounds
- Use maskable safe zone (80% of canvas for important content)
- Background color: #db2777 (theme pink)
- Foreground: White logo/icon
